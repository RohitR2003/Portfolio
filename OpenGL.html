    Retro OpenGL 3D Engine

    This is a work-in-progress retro-style 3D engine built with modern OpenGL. The focus is on learning core graphics programming by building everything from scratch without relying on a full game engine.


    So far, a basic camera system has been implemented, supporting both mouse and keyboard inputs for smooth navigation. The scene currently displays a grid of textured cubes, demonstrating successful rendering with shaders.


    The aim is to eventually support modular rendering, scene management, lighting, and custom materials—all optimized for a retro aesthetic reminiscent of classic 3D games.

    A preview of current progress: textured cube rendering with interactive camera movement.
